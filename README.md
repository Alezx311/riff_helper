## Огляд

GuitarRiff Helper вирішує «творчий блок»: є кілька нот або ідея риффу, але не знаєш куди рухатись далі.

**Можливості:**
- Вводиш ноти на віртуальному грифі → генератор пропонує 5 варіантів продовження або ремікс
- Три інструменти разом: гітара + бас + барабани, синхронне відтворення
- Реальні звукові семпли (гітара, бас, піаніно, струнні, духові тощо)
- Jam Session: граєш у мікрофон — застосунок розпізнає ноти та визначає тональність
- Все в браузері, без серверів і реєстрації

## Головний режим

### Введення нот

#### Вкладки інструментів: Guitar / Bass / Drums

Зелена / помаранчева / синя крапка поряд з назвою вкладки означає, що на цьому інструменті вже є дані.

#### Гітара і бас

Гриф відображає 15 ладів. Фіолетові маркери — ноти поточної гами.

| Дія | Результат |
|-----|-----------|
| Клік по ладу | Додати ноту до поточного біту |
| Клік по тій самій ноті | Видалити ноту з біту |
| `Space` або `Enter` | Завершити біт і почати наступний |
| `→` / `←` | Перемикатись між бітами |
| Клік по чіпу біту | Вибрати біт для редагування |
| `×` на чіпі біту | Видалити біт |
| Клік по тривалості (дробу під бітом) | Змінити тривалість цього біту |

**Акорди:** клікай кілька нот до натискання Space — всі вони потраплять в один біт.

**Бас** ведеться незалежно. Якщо бас-вкладка порожня — генератор автоматично побудує бас-лінію від кореневих нот гітари.

#### Барабани

16-кроковий секвенсор: Kick / Snare / HiHat / Crash / Tom1 / Tom2.
Клік по клітинці вмикає або вимикає удар на цьому кроці.

### Налаштування

| Параметр | Опис |
|----------|------|
| **Key** | Тональність (C–B, 12 варіантів) |
| **Scale** | Гама: Major, Minor, Minor Pentatonic, Major Pentatonic, Blues, Dorian, Phrygian, Harmonic Minor |
| **Tuning** | Строй гітари: Standard, Drop D, DADGAD, Open G, Open D, Half Down, Drop C |
| **Style** | Стиль генерації: Neutral, Rock, Metal, Blues, Jazz, Funk |
| **Generate** | Кількість бітів у результаті: 3, 4, 6, 8, 12 |
| **Mode** | Режим: **Continue** (продовжити фразу) або **Remix** (нова фраза з тих самих нот) |
| **Tempo** | Темп: 40–300 BPM, повзунок + числове поле |
| **Duration** | Тривалість за замовчуванням для нових бітів: 1/2, 1/4, 1/8, 1/16 |

---

### Детектор тональності

Кнопка **`♦ Detect Key/Scale`** з'являється коли є введені ноти.

Натисни — відкриється список тональностей і гам, до яких **входять усі введені ноти**.
Відсортовано від найтіснішого вписування: `+0` означає ідеальний збіг, `+2` — два зайвих ноти гами поза вводом.

Натисни **Apply** поряд з потрібним варіантом — Key і Scale у панелі налаштувань оновляться автоматично.

---

### Генерація варіантів

Кнопки внизу панелі:

| Кнопка | Дія |
|--------|-----|
| **Generate** | Згенерувати 5 варіантів; чекбокси Guitar/Bass/Drums вибирають які інструменти генерувати |
| **Play Input** | Програти введені ноти (всі інструменти разом) |
| **Clear** | Очистити поточний інструмент |
| **Stop** | Зупинити відтворення |

Кожен варіант відображається карткою з:
- Табулатурою (гітара / бас / барабани)
- Оцінкою якості (score)
- Кнопками **Play** (одноразово), **Loop** (у циклі), **Use** (взяти як новий ввід для подальшої генерації)
- Кнопками окремого відтворення кожного інструменту

### Мікшер

Права панель **Mixer**:

| Елемент | Дія |
|---------|-----|
| Повзунок гучності | 0–100% для кожного інструменту |
| **M** | Mute (заглушити) |
| **S** | Solo (залишити тільки цей інструмент) |
| Вибір семплів | Набір семплів: guitar-electric, guitar-acoustic, guitar-nylon, bass-electric, piano, violin, cello, organ, saxophone, bassoon, contrabass |

## Jam Session

Вкладка у верхньому меню. Грай у мікрофон — застосунок розпізнає ноти в реальному часі.

### Запис

1. Обери тональність, гаму і строй
2. Натисни **● REC** — починається запис через мікрофон
3. Грай; поточна нота відображається великим шрифтом; зелений колір — нота в тональності, червоний — поза
4. Натисни **■ STOP** — запис зупиняється

Після зупинки доступні кнопки:
- **→ Генерувати варіанти** — конвертує записані ноти у гітарний ввід і переходить у головний режим
- **↓ Зберегти** — зберігає сесію як JSON-файл

#### Налаштування детектора (AudioDebug)

Розкривна панель з параметрами розпізнавання:

| Параметр | Що регулює |
|----------|-----------|
| FFT Size | Точність частотного аналізу (1024–8192) |
| Smoothing | Згладжування спектру (0–0.99) |
| RMS Threshold | Поріг тиші (нижче — ігнорується) |
| Correlation | Мінімальна якість кореляційного піку |
| Min/Max Freq | Діапазон частот (Hz) |
| Stability Frames | Скільки фреймів нота має бути стабільна перед записом |

---

### Аналіз тональності

Після зупинки запису, якщо є ноти, з'являється блок **«Аналіз тональності»**.

#### Список тональностей

Показує всі тональності й гами, до яких входять розпізнані ноти, відсортовані від найточнішого збігу.
`+N` — скільки нот гами не прозвучало (що менше, то краще вписування).

- Натисни **Застосувати** — тональність і гама встановляться в усьому застосунку
- Натисни на рядок — розгортається деталізація

#### Діатонічні акорди

Для вибраної тональності та гами відображаються всі акорди на ступенях гами:

- Римські цифри (I, ii, iii°, IV, V, vi, vii°)
- Назва акорду (C, Dm, Edim, F, G, Am, Bdim)
- Якість (maj / min / dim / aug)
- Колірне маркування: зелений = мажор, фіолетовий = мінор, червоний = зменшений

#### Прогресії акордів

Типові прогресії для гами з реальними назвами акордів і римськими цифрами:

| Гама | Прогресії |
|------|-----------|
| Major | Pop (I-V-vi-IV), Classic (I-IV-V-I), 50s, II-V-I, Blues |
| Minor | Minor Pop (i-VII-iv-VI), Classic, Andalusian (i-VII-VI-V), II-V-i |
| Dorian | Dorian, Dorian II-V |
| Phrygian | Phrygian, Phryg. Pop |
| Harmonic Minor | HM Classic, HM Jazz |

## Технічний стек

| Компонент | Технологія |
|-----------|-----------|
| UI | React 19 + TypeScript, Vite 8 |
| Стан | useReducer + React Context |
| Звук (гітара/бас) | Tone.js 14 — Sampler з реальними .mp3 семплами |
| Звук (барабани) | Web Audio API — синтез (sine+noise) |
| Розпізнавання нот | Web Audio API — автокореляція з параболічною інтерполяцією |
| Хостинг | GitHub Pages, CI/CD via GitHub Actions |
| Семпли | `public/samples/` (~297MB, 11 наборів інструментів) |

Весь код виконується на стороні клієнта, без серверів і збору даних.

## Структура проекту

```
src/
  audio/           - Tone.Sampler обгортка, Web Audio барабани, React хук відтворення
  components/
    BeatInput/     - Чіпи бітів + вибір тривалості
    Controls/      - Панель налаштувань + детектор тональності
    DrumGrid/      - 16-кроковий секвенсор
    Fretboard/     - Інтерактивний гриф
    GenerationControls/ - Кнопки Generate/Play/Clear/Stop
    Header/        - Шапка + перемикач вкладок
    InstrumentTabs/ - Guitar / Bass / Drums вкладки
    JamSession/    - Запис із мікрофона + аналіз тональності
    MixerPanel/    - Гучність, mute, solo, набір семплів
    Results/       - Картки згенерованих варіантів
  constants/       - Гами, строї, стилі, назви нот
  generation/      - Алгоритм генерації (7 стратегій, scoring, ритм)
  state/           - AppContext, reducer, типи стану
  types/           - Загальні типи (Beat, FretNote, GenerationResult тощо)
  utils/
    midi.ts        - MIDI ↔ нота, перевірка входження в гаму
    keyDetect.ts   - Детектор тональності, діатонічні акорди, прогресії
    tab.ts         - Рендер табулатури
```
